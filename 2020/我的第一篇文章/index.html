Hugo + Github 搭建个人网站指南

Written by EvaL with ♥ on June 29, 2020 in github, blogs, 1011 words

搭建前准备

在使用hugo + GitHub搭建个人网站之前，你需要拥有：

  1. 一个GitHub账号 GitHub

  2. 一台下载了hugo的电脑 hugo下载官方介绍文档

hugo搭建部分

第一步：创建一个新的站点

hugo new site evalyeah

这里的evalyeah是指站点名称，可以修改为自己的站点名称，创建成功后，会创建一个新的叫evalyeah的目录，如下图

![image-20200629205108380](/Users/liuyuhao/Library/Application Support/typora-user-images/pic1.png)

第二步：给站点增加一个主题

这里我以jane主题为例 jane主题

以下分别是指，进入站点目录后，将站点目录git初始化，然后将整个主题克隆至themes文件夹

cd evalyeah
git clone https://github.com/xianmin/hugo-theme-jane.git --depth=1 themes/jane

克隆完成后，我的站点themes目录中会出现jane目录，如下图

![image-20200629210638668](/Users/liuyuhao/Library/Application Support/typora-user-images/pic2.png)

然后我们复制一些主题的示例文本和默认设置

cp -r themes/jane/exampleSite/content ./
cp themes/jane/exampleSite/config.toml ./

复制完后content目录中会出现一些文档

![image-20200629210908733](/Users/liuyuhao/Library/Application Support/typora-user-images/pic3.png)

复制完后config.toml会出现相关默认配置

![image-20200629211029969](/Users/liuyuhao/Library/Application Support/typora-user-images/pic4.png)

第三步：给站点中增加一些内容

hugo new post/Hugo+GitHub搭建个人网站.md

打开此Hugo+GitHub搭建个人网站.md文档，会出现如下基础配置：

---
title: "Hugo+GitHub搭建个人网站"
date: 2020-06-29T21:13:54+08:00
draft: true
---

将draft修改为false（Drafts do not get deployed; once you finish a post, update the header of the post to say draft: false. ），增加一些文字内容

第四步：打开hugo server

hugo server -D

打开后可在http://localhost:1313/中预览网站情况

![image-20200629212015541](/Users/liuyuhao/Library/Application Support/typora-user-images/pic5.png)

第五步：修改基础配置

在config.toml文件中修改相关配置

baseURL = "https://eval.buzz"
languageCode = "en-zh"
title = "EvaL的一些记录"
theme = "jane"

第六步：生成静态页面

hugo -D

成功生成后会产生public目录，如下图

![image-20200629212746879](/Users/liuyuhao/Library/Application Support/typora-user-images/pic6.png)

至此，hugo搭建完毕，接下来使用GitHub pages预览HTML

GitHub托管部分

第一步：新建站点目录仓库

站点目录中新建.gitignore文件中写入/public/

将evalyeah目录push到新的仓库中

第二步：新建public仓库

新建xxx.github.io仓库，并将public目录push到本仓库中

在设置中找到GitHub pages，即可通过GitHub pages预览已生成的静态页面

![image-20200629213808502](/Users/liuyuhao/Library/Application Support/typora-user-images/pic7.png)

![image-20200629213835984](/Users/liuyuhao/Library/Application Support/typora-user-images/pic8.png)

至此，就可以用GitHub pages预览hugo搭建的静态页面啦。

如何发表第二篇文章

第一步：创建一篇新的文章

在站点目录（注意不是public）中运行

hugo new post/第二篇文章.md
hugo -D

第二步：进入public目录发布

然后进入public目录

git add .
git commit -m update
git push -f

过一会儿就能看见自己的第二篇文章啦。
